## COMMAND
Generate *.g.dart files automatically. (MobX)

```
$ flutter packages pub run build_runner watch
or
$ flutter packages pub run build_runner watch --delete-conflicting-outputs
```

Run. (iOS)

```
$ open -a Simulator
$ flutter run
```

Build. (Web)

```
$ flutter build web
(check at local)
$ cd build/web
$ php -S localhost:8080
```
